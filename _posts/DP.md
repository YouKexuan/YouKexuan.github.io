概念
======
随机化算法：对于特定输入，该算法的输出不是固定值，而是服从某一分布

概率单纯形形式化的定义：
\(\Delta(B) = \{ x \in \mathbb{R}^{|B|} : x_i \geq 0 \text{ for all } i \text{ and } \sum_{i=1}^{|B|} x_i = 1 \}\)
